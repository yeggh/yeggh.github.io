Technical Projects

While taking some university courses, I did a number of projects that enhanced my coding skills and teamwork abilities. Also, some of the project required reading and analyzing papers related to my field of research. A few of these projects are described below.

  • Loyalty Club: This project presents a dynamic system to encourage customers to continue using the services of associated businesses. Each business can determine the parameters Score Coefficient, Score Period, and Max Order Score to rank the customers, and place them in different levels. The defined levels offer arbitrary facilities such as free shipping, birthday gifts, or discounts. Also, some analytics plots are offered to present the growth trend of the business.

  • A PyTorch Implemntation of the paper “Graph Convolutional Networks for Text Classification”

  • Decaf Compiler: Our team designed a compiler for the decaf programming language.

  • NRF Comunication: This project presents a two-way communication mechanism implemented on Arduino devices that sends and receives packets using a keypad and a Nokia LCD Display device.

  • Processor with a Complex Number Unit Comprocessor
