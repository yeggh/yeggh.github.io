Research Experience

I am currently researching the following topic:

  • Multi-Sensor Data Fusion with Deep Learning in Image Processing Applications
    In this project, we address the 3D detection task in autonomous driving by exploiting features from multiple sensors, namely 2D camera image and 3D LiDAR point cloud. Multi-Sensor Data Fusion provides a more comprehensive understanding of the environment by combining multiple information obtained from different sensors to avoid the perceptual constraints of a single sensor. For example, camera images usually contain useful features such as color and texture while they lack depth estimations. On the other hand, LiDAR point cloud provides helpful depth information while suffering from being sparse and unordered. However, fusing the representations of LiDAR and camera images, which have different characteristics, is a challenging task.
